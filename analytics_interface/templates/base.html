Learning Measures Explained

Since we don't directly know a student's intent for enrolling in a course, it is often difficult to find simple metrics that demonstrate the impact of our platform on learning. The metrics shown here are one way to get a glimpse into these learning metrics.

Checkmarks - Knowledge

As a proxy for the morsels of knowledge that students are learning, the number of correct answers is a very simple metric that provides a higher bar than the number of students enrolled in a course and more accurate bar than the number of certificates earned. Each correct answer implies that the student has learned something.

Number of Attempts - Effort

Assuming that course authors build assessments with sufficient difficulty, number of attempts serves as a useful proxy for the effort that students put into acquiring morsels of knowledge. Each attempt at solving a problem assumes that the student is putting effort into learning something
